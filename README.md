# EdgeSight-AI
Real-time object detection, scene reasoning, event summarization, and system optimization

Core stack
C++17 or C++20
OpenCV for capture, drawing, preprocessing
ONNX Runtime C++ for inference
or TensorRT later if NVIDIA GPU is available
nlohmann/json for structured scene state
spdlog for logging
CMake for build system
SQLite or JSONL for event storage
REST API or local process call for LLM interface
